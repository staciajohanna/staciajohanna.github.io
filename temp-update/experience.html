MIT EECS Undergraduate Research and Innovation Scholar (SuperUROP)
September 2020 - May 2021 || Cambridge, MA

TypeScript Praxis for Second-Programming-Language Learning

Topic: education technology, programming language learning
Skills used: TypeScript, Visual Studio Code Extension
Abstract While there are many tutoring platforms for people who are learning a programming language for the first time, there are only a few platforms that help experienced programmers learn a new language. Expert programmers can learn a new language by themselves through referencing documentation and relating the new languageâ€™s concepts to those of the old language. However, unguided knowledge transfer between prior and new languages causes false assumptions. TypeScript Praxis is a two-part system that consists of a tutor platform for users with a previous background in Python to learn TypeScript and an interactive guide for problem authors to create exercise code for the tutor platform. The tutor platform provides TypeScript coding exercises in which the problems, hints, and explanations highlight similarities and differences between Python and TypeScript to prevent misconceptions caused by faulty knowledge transfer. The interactive guide helps problem authors in creating TypeScript code examples by semi-automatically reducing the amount of code acquired from public repositories. Case studies of the tutor platform show how users can learn TypeScript concepts with the system, while the case studies for the interactive guide demonstrate that it can be used in creating exercise code for a diverse range of TypeScript concepts.

Spring 2021

The World is Watching: US Election 2020

Topic : data visualization, map visualization, wordcloud, tweets, global
Skills used : d3.js, HTML, CSS, JavaScript
Link to visualization: https://6859-sp21.github.io/a4-theworldiswatching/
Link to Github: https://github.com/6859-sp21/a4-theworldiswatching
Overview The World is Watching: US Election 2020 tells the story of the world intently watching the 2020 U.S. Election. Using tens of thousands of tweets from the days leading up to the election, our visualization makes use of tweet and hashtag counts to answer the question, "What does the world think of the 2020 U.S. Election?" Viewers can explore the dates and hashtags that resulted in the highest (or lowest) number of election-related tweets. Viewers can also zoom in and focus on a specific country or, conversely, watch as the rest of the world discusses the election. With this visualization, we hope viewers can gain a global perspective on such a consequential election in modern American history.
